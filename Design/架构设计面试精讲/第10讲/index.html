    10 еҰӮдҪ•еӣһзӯ” MySQL зҡ„дәӢеҠЎйҡ”зҰ»зә§еҲ«е’Ңй”Ғзҡ„жңәеҲ¶пјҹВ¶

    дёҠдёҖи®ІпјҢжҲ‘и®ІдәҶ MySQL зҡ„зҙўеј•еҺҹзҗҶдёҺдјҳеҢ–й—®йўҳпјҢд»ҠеӨ©жҲ‘еёҰдҪ з»§з»ӯеӯҰд№  MySQL зҡ„дәӢеҠЎйҡ”зҰ»зә§еҲ«е’Ңй”Ғзҡ„жңәеҲ¶пјҢMySQL зҡ„дәӢеҠЎе’Ңй”ҒжҳҜе№¶еҸ‘жҺ§еҲ¶жңҖеҹәжң¬зҡ„жүӢж®өпјҢеңЁйқўиҜ•дёӯпјҢе®ғд»¬дёҺ 09 и®Ізҡ„зҙўеј•дёҖж ·пјҢеҗҢж ·жҳҜ MySQL йҮҚиҰҒзҡ„иҖғеҜҹзӮ№гҖӮ

    жЎҲдҫӢиғҢжҷҜВ¶

    MySQL зҡ„дәӢеҠЎйҡ”зҰ»зә§еҲ«пјҲIsolation LevelпјүпјҢжҳҜжҢҮпјҡеҪ“еӨҡдёӘзәҝзЁӢж“ҚдҪңж•°жҚ®еә“ж—¶пјҢж•°жҚ®еә“иҰҒиҙҹиҙЈйҡ”зҰ»ж“ҚдҪңпјҢжқҘдҝқиҜҒеҗ„дёӘзәҝзЁӢеңЁиҺ·еҸ–ж•°жҚ®ж—¶зҡ„еҮҶзЎ®жҖ§гҖӮе®ғеҲҶдёәеӣӣдёӘдёҚеҗҢзҡ„еұӮж¬ЎпјҢжҢүйҡ”зҰ»ж°ҙе№ій«ҳдҪҺжҺ’еәҸпјҢиҜ»жңӘжҸҗдәӨ < иҜ»е·ІжҸҗдәӨ < еҸҜйҮҚеӨҚеәҰ < дёІиЎҢеҢ–гҖӮ

    MySQL йҡ”зҰ»зә§еҲ«

    • иҜ»жңӘжҸҗдәӨпјҲRead uncommittedпјү пјҡйҡ”зҰ»зә§еҲ«жңҖдҪҺгҖҒйҡ”зҰ»еәҰжңҖејұпјҢи„ҸиҜ»гҖҒдёҚеҸҜйҮҚеӨҚиҜ»гҖҒе№»иҜ»дёүз§ҚзҺ°иұЎйғҪеҸҜиғҪеҸ‘з”ҹгҖӮжүҖд»Ҙе®ғеҹәжң¬жҳҜзҗҶи®әдёҠзҡ„еӯҳеңЁпјҢе®һйҷ…йЎ№зӣ®дёӯжІЎжңүдәәз”ЁпјҢдҪҶжҖ§иғҪжңҖй«ҳгҖӮ
    • иҜ»е·ІжҸҗдәӨпјҲRead committedпјү пјҡе®ғдҝқиҜҒдәҶдәӢеҠЎдёҚеҮәзҺ°дёӯй—ҙзҠ¶жҖҒзҡ„ж•°жҚ®пјҢжүҖжңүж•°жҚ®йғҪжҳҜе·ІжҸҗдәӨдё”жӣҙж–°зҡ„пјҢи§ЈеҶідәҶи„ҸиҜ»зҡ„й—®йўҳгҖӮдҪҶиҜ»е·ІжҸҗдәӨзә§еҲ«дҫқж—§еҫҲдҪҺпјҢе®ғе…Ғи®ёдәӢеҠЎй—ҙеҸҜе№¶еҸ‘дҝ®ж”№ж•°жҚ®пјҢжүҖд»ҘдёҚдҝқиҜҒеҶҚж¬ЎиҜ»еҸ–ж—¶иғҪеҫ—еҲ°еҗҢж ·зҡ„ж•°жҚ®пјҢд№ҹе°ұжҳҜиҝҳдјҡеӯҳеңЁдёҚеҸҜйҮҚеӨҚиҜ»гҖҒе№»иҜ»зҡ„еҸҜиғҪгҖӮ
    • еҸҜйҮҚеӨҚиҜ»пјҲRepeatable readsпјү пјҡMySQL InnoDB еј•ж“Һзҡ„й»ҳи®Өйҡ”зҰ»зә§еҲ«пјҢдҝқиҜҒеҗҢдёҖдёӘдәӢеҠЎдёӯеӨҡж¬ЎиҜ»еҸ–ж•°жҚ®зҡ„дёҖиҮҙжҖ§пјҢи§ЈеҶіи„ҸиҜ»е’ҢдёҚеҸҜйҮҚеӨҚиҜ»пјҢдҪҶд»Қз„¶еӯҳеңЁе№»иҜ»зҡ„еҸҜиғҪгҖӮ
    • еҸҜдёІиЎҢеҢ–пјҲSerializableпјү пјҡйҖүжӢ©вҖңеҸҜдёІиЎҢеҢ–вҖқж„Ҹе‘ізқҖиҜ»еҸ–ж•°жҚ®ж—¶пјҢйңҖиҰҒиҺ·еҸ–е…ұдә«иҜ»й”Ғпјӣжӣҙж–°ж•°жҚ®ж—¶пјҢйңҖиҰҒиҺ·еҸ–жҺ’д»–еҶҷй”ҒпјӣеҰӮжһң SQL дҪҝз”Ё WHERE иҜӯеҸҘпјҢиҝҳдјҡиҺ·еҸ–еҢәй—ҙй”ҒгҖӮжҚўеҸҘиҜқиҜҙпјҢдәӢеҠЎ A ж“ҚдҪңж•°жҚ®еә“ж—¶пјҢдәӢеҠЎ B еҸӘиғҪжҺ’йҳҹзӯүеҫ…пјҢеӣ жӯӨжҖ§иғҪд№ҹжңҖдҪҺгҖӮ

    иҮідәҺж•°жҚ®еә“й”ҒпјҢеҲҶдёәжӮІи§Ӯй”Ғе’Ңд№җи§Ӯй”ҒпјҢвҖңжӮІи§Ӯй”ҒвҖқи®Өдёәж•°жҚ®еҮәзҺ°еҶІзӘҒзҡ„еҸҜиғҪжҖ§еҫҲеӨ§пјҢвҖңд№җи§Ӯй”ҒвҖқи®Өдёәж•°жҚ®еҮәзҺ°еҶІзӘҒзҡ„еҸҜиғҪжҖ§дёҚеӨ§гҖӮйӮЈжӮІи§Ӯй”Ғе’Ңд№җи§Ӯй”ҒеңЁеҹәдәҺ MySQL ж•°жҚ®еә“зҡ„еә”з”ЁејҖеҸ‘дёӯпјҢжҳҜеҰӮдҪ•е®һзҺ°зҡ„е‘ўпјҹ

    • жӮІи§Ӯй”ҒдёҖиҲ¬еҲ©з”Ё SELECT вҖҰ FOR UPDATE зұ»дјјзҡ„иҜӯеҸҘпјҢеҜ№ж•°жҚ®еҠ й”ҒпјҢйҒҝе…Қе…¶д»–дәӢеҠЎж„ҸеӨ–дҝ®ж”№ж•°жҚ®гҖӮ
    • д№җи§Ӯй”ҒеҲ©з”Ё CAS жңәеҲ¶пјҢе№¶дёҚдјҡеҜ№ж•°жҚ®еҠ й”ҒпјҢиҖҢжҳҜйҖҡиҝҮеҜ№жҜ”ж•°жҚ®зҡ„ж—¶й—ҙжҲіжҲ–иҖ…зүҲжң¬еҸ·пјҢе®һзҺ°зүҲжң¬еҲӨж–ӯгҖӮ

    жЎҲдҫӢеҲҶжһҗВ¶

    еҰӮжһңйқўиҜ•е®ҳжғіж·ұжҢ–еҖҷйҖүдәәеҜ№ж•°жҚ®еә“еҶ…йғЁжңәеҲ¶зҡ„жҺҢжҸЎзЁӢеәҰпјҢеҲҮе…ҘзӮ№дёҖиҲ¬жҳҜ MySQL зҡ„дәӢеҠЎе’Ңй”ҒжңәеҲ¶гҖӮжҺҘдёӢжқҘпјҢжҲ‘е°ұд»ҺеҲқдёӯзә§з ”еҸ‘е·ҘзЁӢеёҲзҡ„и§’еәҰеҮәеҸ‘пјҢд»ҺжҰӮеҝөеҲ°е®һи·өпјҢеёҰдҪ жҺҢжҸЎвҖңMySQL дәӢеҠЎе’Ңй”ҒжңәеҲ¶вҖқзҡ„й«ҳйў‘иҖғзӮ№пјҡ

    • дёҫдҫӢиҜҙжҳҺд»Җд№ҲжҳҜи„ҸиҜ»гҖҒдёҚеҸҜйҮҚеӨҚеәҰе’Ңе№»иҜ»пјҲдёүиҖ…иҷҪз„¶еҹәзЎҖпјҢдҪҶеҫҲеӨҡеҗҢеӯҰе®№жҳ“еј„ж··пјүпјҹ
    • MySQL жҳҜжҖҺд№Ҳи§ЈеҶіи„ҸиҜ»гҖҒдёҚеҸҜйҮҚеӨҚиҜ»пјҢе’Ңе№»иҜ»й—®йўҳзҡ„пјҹ
    • дҪ жҖҺд№ҲзҗҶи§Јжӯ»й”Ғпјҹ
    • вҖҰвҖҰ

    жЎҲдҫӢи§Јзӯ”В¶

    жҖҺд№ҲзҗҶи§Ји„ҸиҜ»гҖҒдёҚеҸҜйҮҚеӨҚиҜ»е’Ңе№»иҜ»пјҹВ¶

    и„ҸиҜ»пјҡ иҜ»еҲ°дәҶжңӘжҸҗдәӨдәӢеҠЎзҡ„ж•°жҚ®гҖӮ

    дәӢеҠЎе№¶еҸ‘ж—¶зҡ„вҖңи„ҸиҜ»вҖқзҺ°иұЎ

    еҒҮи®ҫжңү A е’Ң B дёӨдёӘдәӢеҠЎпјҢеңЁе№¶еҸ‘жғ…еҶөдёӢпјҢдәӢеҠЎ A е…ҲејҖе§ӢиҜ»еҸ–е•Ҷе“Ғж•°жҚ®иЎЁдёӯзҡ„ж•°жҚ®пјҢз„¶еҗҺеҶҚжү§иЎҢжӣҙж–°ж“ҚдҪңпјҢеҰӮжһңжӯӨж—¶дәӢеҠЎ A иҝҳжІЎжңүжҸҗдәӨжӣҙж–°ж“ҚдҪңпјҢдҪҶжҒ°еҘҪдәӢеҠЎ B ејҖе§ӢпјҢз„¶еҗҺд№ҹйңҖиҰҒиҜ»еҸ–е•Ҷе“Ғж•°жҚ®пјҢжӯӨж—¶дәӢеҠЎ B жҹҘиҜўеҫ—еҲ°зҡ„жҳҜеҲҡжүҚдәӢеҠЎ A жӣҙж–°еҗҺзҡ„ж•°жҚ®гҖӮ

    еҰӮжһңжҺҘдёӢжқҘдәӢеҠЎ A и§ҰеҸ‘дәҶеӣһж»ҡпјҢйӮЈд№ҲдәӢеҠЎ B еҲҡжүҚиҜ»еҲ°зҡ„ж•°жҚ®е°ұжҳҜиҝҮж—¶зҡ„ж•°жҚ®пјҢиҝҷз§ҚзҺ°иұЎе°ұжҳҜи„ҸиҜ»гҖӮвҖңи„ҸиҜ»вҖқйқўиҜ•е…іжіЁзӮ№пјҡ 1. и„ҸиҜ»еҜ№еә”зҡ„йҡ”зҰ»зә§еҲ«жҳҜвҖңиҜ»жңӘжҸҗдәӨвҖқпјҢеҸӘжңүиҜҘйҡ”зҰ»зә§еҲ«жүҚдјҡеҮәзҺ°и„ҸиҜ»гҖӮ 2. и„ҸиҜ»зҡ„и§ЈеҶіеҠһжі•жҳҜеҚҮзә§дәӢеҠЎйҡ”зҰ»зә§еҲ«пјҢжҜ”еҰӮвҖңиҜ»е·ІжҸҗдәӨвҖқгҖӮдёҚеҸҜйҮҚеӨҚиҜ»пјҡ дәӢеҠЎ A е…ҲиҜ»еҸ–дёҖжқЎж•°жҚ®пјҢз„¶еҗҺжү§иЎҢйҖ»иҫ‘зҡ„иҝҮзЁӢдёӯпјҢдәӢеҠЎ B жӣҙж–°дәҶиҝҷжқЎж•°жҚ®пјҢдәӢеҠЎ A еҶҚиҜ»еҸ–ж—¶пјҢеҸ‘зҺ°ж•°жҚ®дёҚеҢ№й…ҚпјҢиҝҷдёӘзҺ°иұЎе°ұжҳҜвҖңдёҚеҸҜйҮҚеӨҚиҜ»вҖқгҖӮ

    дәӢеҠЎе№¶еҸ‘ж—¶зҡ„вҖңдёҚеҸҜйҮҚеӨҚиҜ»вҖқзҺ°иұЎ вҖңдёҚеҸҜйҮҚеӨҚиҜ»вҖқйқўиҜ•е…іжіЁзӮ№пјҡ 1. з®ҖеҚ•зҗҶи§ЈжҳҜдёӨж¬ЎиҜ»еҸ–зҡ„ж•°жҚ®дёӯй—ҙиў«дҝ®ж”№пјҢеҜ№еә”зҡ„йҡ”зҰ»зә§еҲ«жҳҜвҖңиҜ»жңӘжҸҗдәӨвҖқжҲ–вҖңиҜ»е·ІжҸҗдәӨвҖқгҖӮ 2. дёҚеҸҜйҮҚеӨҚиҜ»зҡ„и§ЈеҶіеҠһжі•е°ұжҳҜеҚҮзә§дәӢеҠЎйҡ”зҰ»зә§еҲ«пјҢжҜ”еҰӮвҖңеҸҜйҮҚеӨҚеәҰвҖқгҖӮе№»иҜ»пјҡ еңЁдёҖдёӘдәӢеҠЎеҶ…пјҢеҗҢдёҖжқЎжҹҘиҜўиҜӯеҸҘеңЁдёҚеҗҢж—¶й—ҙж®өжү§иЎҢпјҢеҫ—еҲ°дёҚеҗҢзҡ„з»“жһңйӣҶгҖӮ

    дәӢеҠЎе№¶еҸ‘ж—¶зҡ„вҖңе№»иҜ»вҖқзҺ°иұЎ

    дәӢеҠЎ A иҜ»дәҶдёҖж¬Ўе•Ҷе“ҒиЎЁпјҢеҫ—еҲ°жңҖеҗҺзҡ„ ID жҳҜ 3пјҢдәӢеҠЎ B д№ҹеҗҢж ·иҜ»дәҶдёҖж¬ЎпјҢеҫ—еҲ°жңҖеҗҺ ID д№ҹжҳҜ 3гҖӮжҺҘдёӢжқҘдәӢеҠЎ A е…ҲжҸ’е…ҘдәҶдёҖиЎҢпјҢз„¶еҗҺиҜ»дәҶдёҖдёӢжңҖж–°зҡ„ ID жҳҜ 4пјҢеҲҡеҘҪжҳҜеүҚйқў ID 3 еҠ дёҠ 1пјҢз„¶еҗҺдәӢеҠЎ B д№ҹжҸ’е…ҘдәҶдёҖиЎҢпјҢжҺҘзқҖиҜ»дәҶдёҖдёӢжңҖж–°зҡ„ ID еҸ‘зҺ°жҳҜ 5пјҢиҖҢдёҚжҳҜ 3 еҠ  1гҖӮ

    иҝҷж—¶пјҢдҪ еҸ‘зҺ°еңЁдҪҝз”Ё ID еҒҡеҲӨж–ӯжҲ–еҒҡе…ій”®ж•°жҚ®ж—¶пјҢе°ұдјҡеҮәзҺ°й—®йўҳпјҢиҝҷз§ҚзҺ°иұЎе°ұеғҸжҳҜи®©дәӢеҠЎ B дә§з”ҹдәҶе№»и§үдёҖж ·пјҢиҜ»еҸ–еҲ°дәҶдёҖдёӘж„ҸжғідёҚеҲ°зҡ„ж•°жҚ®пјҢжүҖд»ҘеҸ«е№»иҜ»гҖӮеҪ“з„¶пјҢдёҚд»…д»…жҳҜж–°еўһпјҢеҲ йҷӨгҖҒдҝ®ж”№ж•°жҚ®д№ҹдјҡеҸ‘з”ҹзұ»дјјзҡ„жғ…еҶөгҖӮвҖңе№»иҜ»вҖқйқўиҜ•е…іжіЁзӮ№пјҡ 1. иҰҒжғіи§ЈеҶіе№»иҜ»дёҚиғҪеҚҮзә§дәӢеҠЎйҡ”зҰ»зә§еҲ«еҲ°вҖңеҸҜдёІиЎҢеҢ–вҖқпјҢйӮЈж ·ж•°жҚ®еә“д№ҹеӨұеҺ»дәҶе№¶еҸ‘еӨ„зҗҶиғҪеҠӣгҖӮ 2. иЎҢй”Ғи§ЈеҶідёҚдәҶе№»иҜ»пјҢеӣ дёәеҚідҪҝй”ҒдҪҸжүҖжңүи®°еҪ•пјҢиҝҳжҳҜйҳ»жӯўдёҚдәҶжҸ’е…Ҙж–°ж•°жҚ®гҖӮ 3. и§ЈеҶіе№»иҜ»зҡ„еҠһжі•жҳҜй”ҒдҪҸи®°еҪ•д№Ӣй—ҙзҡ„вҖңй—ҙйҡҷвҖқпјҢдёәжӯӨ MySQL InnoDB еј•е…ҘдәҶж–°зҡ„й”ҒпјҢеҸ« й—ҙйҡҷй”ҒпјҲGap LockпјүпјҢжүҖд»ҘеңЁйқўиҜ•дёӯпјҢдҪ д№ҹиҰҒжҺҢжҸЎй—ҙйҡҷй”ҒпјҢд»ҘеҸҠй—ҙйҡҷй”ҒдёҺиЎҢй”Ғз»“еҗҲзҡ„ next-key lock й”ҒгҖӮ

    жҖҺд№ҲзҗҶи§Јжӯ»й”ҒВ¶

    йҷӨдәҶдәӢеҠЎйҡ”зҰ»зә§еҲ«пјҢеҫҲеӨҡеҗҢеӯҰеңЁйқўиҜ•ж—¶пјҢз»Ҹеёёдјҡиў«йқўиҜ•е®ҳзӣҙеҘ”дё»йўҳең°й—®пјҡвҖңи°Ҳи°ҲдҪ еҜ№жӯ»й”Ғзҡ„зҗҶи§ЈвҖқгҖӮиҰҒеӣһзӯ”иҝҷж ·ејҖж”ҫзҡ„й—®йўҳпјҢдҪ е°ұиҰҒеңЁи„‘жө·дёӯжўізҗҶеҮәзі»з»ҹеҢ–зҡ„еӣһзӯ”жҖқи·Ҝпјҡ жӯ»й”ҒжҳҜеҰӮдҪ•дә§з”ҹзҡ„пјҢеҰӮдҪ•йҒҝе…Қжӯ»й”ҒгҖӮ жӯ»й”ҒдёҖиҲ¬еҸ‘з”ҹеңЁеӨҡзәҝзЁӢпјҲдёӨдёӘжҲ–дёӨдёӘд»ҘдёҠпјүжү§иЎҢзҡ„иҝҮзЁӢдёӯгҖӮеӣ дёәдәүеӨәиө„жәҗйҖ жҲҗзәҝзЁӢд№Ӣй—ҙзӣёдә’зӯүеҫ…пјҢиҝҷз§Қжғ…еҶөе°ұдә§з”ҹдәҶжӯ»й”ҒгҖӮжҲ‘еңЁ 06 и®Ід№ҹжҸҗеҲ°дәҶжӯ»й”ҒпјҢдҪҶжҳҜе№¶жІЎжңүи®Іе®ғдә§з”ҹзҡ„еҺҹеӣ д»ҘеҸҠжҖҺд№ҲйҒҝе…ҚпјҢжүҖд»ҘжҺҘдёӢжқҘжҲ‘д»¬е°ұжқҘдәҶи§ЈиҝҷйғЁеҲҶеҶ…е®№гҖӮ

    зәҝзЁӢжӯ»й”Ғ

    жҜ”еҰӮдҪ жңүиө„жәҗ 1 е’Ң 2пјҢд»ҘеҸҠзәҝзЁӢ A е’Ң BпјҢеҪ“зәҝзЁӢ A еңЁе·Із»ҸиҺ·еҸ–еҲ°иө„жәҗ 1 зҡ„жғ…еҶөдёӢпјҢжңҹжңӣиҺ·еҸ–зәҝзЁӢ B жҢҒжңүзҡ„иө„жәҗ 2гҖӮдёҺжӯӨеҗҢж—¶пјҢзәҝзЁӢ B еңЁе·Із»ҸиҺ·еҸ–еҲ°иө„жәҗ 2 зҡ„жғ…еҶөдёӢпјҢжңҹжңӣиҺ·еҸ–зҺ°еңә A жҢҒжңүзҡ„иө„жәҗ 1гҖӮ

    йӮЈд№ҲзәҝзЁӢ A е’ҢзәҝзЁӢ B е°ұеӨ„зҗҶдәҶзӣёдә’зӯүеҫ…зҡ„жӯ»й”ҒзҠ¶жҖҒпјҢеңЁжІЎжңүеӨ–еҠӣе№Ійў„зҡ„жғ…еҶөдёӢпјҢзәҝзЁӢ A е’ҢзәҝзЁӢ B е°ұдјҡдёҖзӣҙеӨ„дәҺзӣёдә’зӯүеҫ…зҡ„зҠ¶жҖҒпјҢд»ҺиҖҢдёҚиғҪеӨ„зҗҶе…¶д»–зҡ„иҜ·жұӮгҖӮжӯ»й”Ғдә§з”ҹзҡ„еӣӣдёӘеҝ…иҰҒжқЎд»¶гҖӮ

    дә’ж–ҘжқЎд»¶ дә’ж–Ҙпјҡ еӨҡдёӘзәҝзЁӢдёҚиғҪеҗҢж—¶дҪҝз”ЁдёҖдёӘиө„жәҗгҖӮжҜ”еҰӮзәҝзЁӢ A е·Із»ҸжҢҒжңүзҡ„иө„жәҗпјҢдёҚиғҪеҶҚеҗҢж—¶иў«зәҝзЁӢ B жҢҒжңүгҖӮеҰӮжһңзәҝзЁӢ B иҜ·жұӮиҺ·еҸ–зәҝзЁӢ A е·Із»ҸеҚ жңүзҡ„иө„жәҗпјҢйӮЈзәҝзЁӢ B еҸӘиғҪзӯүеҫ…иҝҷдёӘиө„жәҗиў«зәҝзЁӢ A йҮҠж”ҫгҖӮ

    жҢҒжңүе№¶зӯүеҫ… жҢҒжңүе№¶зӯүеҫ…пјҡ еҪ“зәҝзЁӢ A е·Із»ҸжҢҒжңүдәҶиө„жәҗ 1пјҢеҸҲжҸҗеҮәз”іиҜ·иө„жәҗ 2пјҢдҪҶжҳҜиө„жәҗ 2 е·Із»Ҹиў«зәҝзЁӢ C еҚ з”ЁпјҢжүҖд»ҘзәҝзЁӢ A е°ұдјҡеӨ„дәҺзӯүеҫ…зҠ¶жҖҒпјҢдҪҶе®ғеңЁзӯүеҫ…иө„жәҗ 2 зҡ„еҗҢж—¶е№¶дёҚдјҡйҮҠж”ҫиҮӘе·ұе·Із»ҸиҺ·еҸ–зҡ„иө„жәҗ 1гҖӮ

    дёҚеҸҜеүҘеӨәжқЎд»¶ дёҚеҸҜеүҘеӨәпјҡ зәҝзЁӢ A иҺ·еҸ–еҲ°иө„жәҗ 1 д№ӢеҗҺпјҢеңЁиҮӘе·ұдҪҝз”Ёе®Ңд№ӢеүҚдёҚиғҪиў«е…¶д»–зәҝзЁӢпјҲжҜ”еҰӮзәҝзЁӢ BпјүжҠўеҚ дҪҝз”ЁгҖӮеҰӮжһңзәҝзЁӢ B д№ҹжғідҪҝз”Ёиө„жәҗ 1пјҢеҸӘиғҪеңЁзәҝзЁӢ A дҪҝз”Ёе®ҢеҗҺпјҢдё»еҠЁйҮҠж”ҫеҗҺеҶҚиҺ·еҸ–гҖӮ

    еҫӘзҺҜзӯүеҫ… еҫӘзҺҜзӯүеҫ…пјҡ еҸ‘з”ҹжӯ»й”Ғж—¶пјҢеҝ…з„¶дјҡеӯҳеңЁдёҖдёӘзәҝзЁӢпјҢд№ҹе°ұжҳҜиө„жәҗзҡ„зҺҜеҪўй“ҫгҖӮжҜ”еҰӮзәҝзЁӢ A е·Із»ҸиҺ·еҸ–дәҶиө„жәҗ 1пјҢдҪҶеҗҢж—¶еҸҲиҜ·жұӮиҺ·еҸ–иө„жәҗ 2гҖӮзәҝзЁӢ B е·Із»ҸиҺ·еҸ–дәҶиө„жәҗ 2пјҢдҪҶеҗҢж—¶еҸҲиҜ·жұӮиҺ·еҸ–иө„жәҗ 1пјҢиҝҷе°ұдјҡеҪўжҲҗдёҖдёӘзәҝзЁӢе’Ңиө„жәҗиҜ·жұӮзӯүеҫ…зҡ„зҺҜеҪўеӣҫгҖӮ

    жӯ»й”ҒеҸӘжңүеҗҢж—¶ж»Ўи¶і дә’ж–Ҙ гҖҒ жҢҒжңүе№¶зӯүеҫ… гҖҒ дёҚеҸҜеүҘеӨә гҖҒ еҫӘзҺҜзӯүеҫ… ж—¶жүҚдјҡеҸ‘з”ҹгҖӮе№¶еҸ‘еңәжҷҜдёӢдёҖж—Ұжӯ»й”ҒпјҢдёҖиҲ¬жІЎжңүзү№еҲ«еҘҪзҡ„ж–№жі•пјҢеҫҲеӨҡж—¶еҖҷеҸӘиғҪйҮҚеҗҜеә”з”ЁгҖӮеӣ жӯӨпјҢжңҖеҘҪжҳҜи§„йҒҝжӯ»й”ҒпјҢйӮЈд№Ҳе…·дҪ“жҖҺд№ҲеҒҡе‘ўпјҹзӯ”жЎҲжҳҜпјҡиҮіе°‘з ҙеқҸе…¶дёӯдёҖдёӘжқЎд»¶пјҲдә’ж–Ҙеҝ…йЎ»ж»Ўи¶іпјҢдҪ еҸҜд»Ҙд»Һе…¶д»–дёүдёӘжқЎд»¶еҮәеҸ‘пјүгҖӮ

    • жҢҒжңүе№¶зӯүеҫ…пјҡжҲ‘д»¬еҸҜд»ҘдёҖж¬ЎжҖ§з”іиҜ·жүҖжңүзҡ„иө„жәҗпјҢиҝҷж ·е°ұдёҚеӯҳеңЁзӯүеҫ…дәҶгҖӮ
    • дёҚеҸҜеүҘеӨәпјҡеҚ з”ЁйғЁеҲҶиө„жәҗзҡ„зәҝзЁӢиҝӣдёҖжӯҘз”іиҜ·е…¶д»–иө„жәҗж—¶пјҢеҰӮжһңз”іиҜ·дёҚеҲ°пјҢеҸҜд»Ҙдё»еҠЁйҮҠж”ҫе®ғеҚ жңүзҡ„иө„жәҗпјҢиҝҷж ·дёҚеҸҜеүҘеӨәиҝҷдёӘжқЎд»¶е°ұз ҙеқҸжҺүдәҶгҖӮ
    • еҫӘзҺҜзӯүеҫ…пјҡеҸҜд»Ҙйқ жҢүеәҸз”іиҜ·иө„жәҗжқҘйў„йҳІпјҢд№ҹе°ұжҳҜжүҖи°“зҡ„иө„жәҗжңүеәҸеҲҶй…ҚеҺҹеҲҷпјҢи®©иө„жәҗзҡ„з”іиҜ·е’ҢдҪҝз”ЁжңүзәҝжҖ§йЎәеәҸпјҢз”іиҜ·зҡ„ж—¶еҖҷеҸҜд»Ҙе…Ҳз”іиҜ·иө„жәҗеәҸеҸ·е°Ҹзҡ„пјҢеҶҚз”іиҜ·иө„жәҗеәҸеҸ·еӨ§зҡ„пјҢиҝҷж ·зҡ„зәҝжҖ§еҢ–ж“ҚдҪңе°ұиҮӘз„¶е°ұдёҚеӯҳеңЁеҫӘзҺҜдәҶгҖӮ

    жҖ»з»“В¶

    жҲ‘д»¬иҠұдәҶдёӨи®Ізҡ„ж—¶й—ҙпјҢжҠҠ MySQL ж•°жҚ®еә“йқўиҜ•дёӯзҡ„й«ҳйў‘й—®йўҳзҶҹжӮүдәҶдёҖйҒҚпјҢдҪҶжҳҜеҰӮжһңд»Һж•°жҚ®еә“йўҶеҹҹеә”з”ЁејҖеҸ‘иҖ…и§’еәҰеҮәеҸ‘пјҢиҮіе°‘иҝҳйңҖиҰҒжҺҢжҸЎд»ҘдёӢеҮ йғЁеҲҶеҶ…е®№гҖӮ

    • ж•°жҚ®еә“и®ҫи®ЎеҹәзЎҖ пјҡжҺҢжҸЎж•°жҚ®еә“и®ҫи®Ўдёӯзҡ„еҹәжң¬иҢғејҸпјҢд»ҘеҸҠеҹәзЎҖжҰӮеҝөпјҢдҫӢеҰӮиЎЁгҖҒи§ҶеӣҫгҖҒзҙўеј•гҖҒеӨ–й”®гҖҒеәҸеҲ—еҸ·з”ҹжҲҗеҷЁзӯүпјҢжҺҢжҸЎж•°жҚ®еә“зҡ„ж•°жҚ®зұ»еһӢзҡ„дҪҝз”ЁпјҢжё…жҘҡдёҡеҠЎе®һдҪ“е…ізі»дёҺж•°жҚ®еә“з»“жһ„зҡ„жҳ е°„гҖӮ
    • ж•°жҚ®еә“йҡ”зҰ»зә§еҲ« пјҡжҺҢжҸЎ MySQL еӣӣз§ҚдәӢеҠЎйҡ”зҰ»зә§еҲ«зҡ„еҹәзЎҖзҹҘиҜҶпјҢе№¶иҝӣдёҖжӯҘдәҶи§Ј MVCCгҖҒLocking зӯүжңәеҲ¶еҜ№дәҺеӨ„зҗҶзҡ„иҝӣйҳ¶й—®йўҳзҡ„и§ЈеҶіпјӣиҝҳйңҖиҰҒдәҶи§ЈдёҚеҗҢзҙўеј•зұ»еһӢзҡ„дҪҝз”ЁпјҢз”ҡиҮіжҳҜеә•еұӮж•°жҚ®з»“жһ„е’Ңз®—жі•зӯүгҖӮ
    • SQL дјҳеҢ– пјҡжҺҢжҸЎеҹәзЎҖзҡ„ SQL и°ғдјҳжҠҖе·§пјҢиҮіе°‘иҰҒдәҶи§Јеҹәжң¬жҖқи·ҜжҳҜжҖҺж ·зҡ„пјҢдҫӢеҰӮ SQL жҖҺж ·еҶҷжүҚиғҪжӣҙеҘҪеҲ©з”Ёзҙўеј•гҖҒзҹҘйҒ“еҰӮдҪ•еҲҶжһҗ SQL жү§иЎҢи®ЎеҲ’зӯүгҖӮ
    • ж•°жҚ®еә“жһ¶жһ„и®ҫи®Ў пјҡжҺҢжҸЎй’ҲеҜ№й«ҳе№¶еҸ‘зӯүзү№е®ҡеңәжҷҜдёӯзҡ„и§ЈеҶіж–№жЎҲпјҢеҰӮиҜ»еҶҷеҲҶзҰ»гҖҒеҲҶеә“еҲҶиЎЁзӯүгҖӮ

    еҪ“з„¶еңЁеҮҶеӨҮйқўиҜ•ж—¶жҲ‘е№¶дёҚе»әи®®дҪ жүҫдёҖе Ҷд№Ұй—·еӨҙиӢҰиҜ»пјҢиҝҳжҳҜиҰҒд»Һе®һйҷ…е·ҘдҪңдёӯпјҢд»ҺдҪҝз”Ёж•°жҚ®еә“еҮәеҸ‘пјҢе№¶з»“еҗҲе®һи·өпјҢе®Ңе–„е’Ңж·ұеҢ–иҮӘе·ұзҡ„зҹҘиҜҶдҪ“зі»пјҢд»ҠеӨ©зҡ„еҶ…е®№е°ұи®ІеҲ°иҝҷйҮҢпјҢжҲ‘д»¬дёӢдёҖи®Іи§ҒгҖӮ
